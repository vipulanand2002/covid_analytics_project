## ğŸš€ Installation & Setup

### Prerequisites
- **Power BI Desktop** (Free) - [Download here](https://powerbi.microsoft.com/desktop/)
- **Microsoft Account** (for Power BI Service publishing)

### Steps to Use

1. **Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/covid19-india-dashboard.git
   cd covid19-india-dashboard
   ```

2. **Open the Dashboard**
   - Locate the `.pbix` file in the repository
   - Double-click to open in Power BI Desktop

3. **Refresh Data (Optional)**
   - Click "Refresh" in Power BI Desktop
   - Ensure data files are in the `/data/` folder

4. **Explore the Dashboard**
   - Navigate between pages using tabs at bottom
   - Use filters and slicers for custom analysis
   - Click on visuals for cross-filtering

### Publishing to Power BI Service

1. Click **"Publish"** in Power BI Desktop
2. Sign in to your Microsoft account
3. Select your workspace
4. Access online at [app.powerbi.com](https://app.powerbi.com)
